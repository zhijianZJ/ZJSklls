# ZJSkills 使用场景与提示词

下面的提示词用于启动或继续学习系统设计。示例结果只是可能产物，不代表对岗位、收入、晋升或项目结果的保证。使用时请补充真实约束，并让 Skill 明确区分自述、证据、推断和假设。

## 场景一：零基础了解 AI 行业

### 首次提示词

```text
请使用 ZJSkills 帮我在 4 周内形成对 AI 行业的结构化理解，并判断我更适合继续探索技术、产品还是业务应用方向。我目前没有编程经验，每周 5 小时，预算 300 元。请先问会影响方向判断的关键问题，不要直接给课程清单。
```

### Skill 可能追问什么

会追问你当前行业、过去解决过的问题、是否愿意编程、英语阅读、可获得的设备，以及 4 周后希望用什么证据做方向选择。

### 可能形成的产物

AI 行业角色地图、可迁移能力假设、三个小型体验任务、方向决策评分表、4 周探索路线与每周证据清单。

### 第二轮继续提示词

```text
我更喜欢把模糊需求变成方案，不排斥轻量技术，但暂不想成为全职工程师。请据此生成第一版能力探索图、三个最小体验项目和本周计划，并标出哪些结论仍是假设。
```

### 学习后的复盘与重规划提示词

```text
我完成了三个体验任务，产品方案得分最高，但自动化任务最有投入感。这里是产物和复盘：[内容]。请更新方向判断，保留冲突证据，并设计下一轮最小区分性实验。
```

## 场景二：转岗 AI Agent 工程师

### 首次提示词

```text
请使用 ZJSkills 评估我转岗 AI Agent 工程师的可行路线。我的背景是 3 年数据分析，会 SQL 和少量 Python，每周 10 小时，希望 8 个月后具备用真实项目参与岗位评估的能力。请先定义成功证据、识别关键差距和约束，不要用学完课程代替能力判断。
```

### Skill 可能追问什么

会追问 Python 与工程实践证据、API/Git/部署经验、目标岗位样本、可公开数据、云资源预算、英语文档阅读与期限弹性。

### 可能形成的产物

L0-L5 能力树、Python/API/LLM/RAG/工具调用/工作流/评估部署依赖图，六级项目阶梯，能力覆盖矩阵、阶段路线和独立交付评分表。

### 第二轮继续提示词

```text
这是我现有的 Python 仓库、测试结果和目标岗位描述：[内容]。请只根据可核验证据评估当前等级，生成优先差距、前三个项目和第一周最低可交付计划。
```

### 学习后的复盘与重规划提示词

```text
RAG 项目能运行，但在需求变化后检索质量下降，我无法独立定位问题。请按项目证据评分，找出最早因果缺口，增加从指导调试到独立调试的练习，并更新后续路线。
```

## 场景三：转岗 AI 产品经理

### 首次提示词

```text
请使用 ZJSkills 为我设计 AI 产品经理转岗系统。我有 5 年电商运营经验，没有开发经历，每周 8 小时，目标是 6 个月内形成可被目标团队评估的 AI 产品案例与工作样本。请先分析我的可迁移能力、技术理解缺口和目标岗位差异。
```

### Skill 可能追问什么

会追问具体目标行业、需求分析与数据决策证据、是否做过跨团队交付、技术沟通基础、可访问用户、可用于验证的问题及作品公开限制。

### 可能形成的产物

业务问题定义、AI 能力边界、数据与评估、原型验证、技术沟通、交付协作等能力模型；从问题验证到可测试原型的项目阶梯；案例文档与演示评分标准。

### 第二轮继续提示词

```text
目标是电商客服与运营效率方向。这是我过去的需求文档和数据分析案例：[内容]。请判断哪些能力已经有证据、哪些只是迁移假设，并生成一个真实用户问题驱动的首个项目简报。
```

### 学习后的复盘与重规划提示词

```text
用户访谈证明问题存在，但原型没有改善关键指标，研发反馈需求边界不清。请保留这些负面证据，重新评估问题定义、验收指标和技术沟通能力，并调整项目而不是直接增加更多知识内容。
```

## 场景四：在当前工作中应用 AI

### 首次提示词

```text
请使用 ZJSkills 帮我在现岗位中落地一个 AI 应用。我负责制造业采购，每周可投入 4 小时，希望 10 周内减少供应商资料整理的重复劳动，同时遵守公司数据与审批要求。请先澄清业务价值、风险、权限和可验证指标。
```

### Skill 可能追问什么

会追问当前流程耗时与错误率、数据敏感级别、工具权限、审批人、失败影响、可用样本、人工复核点和成功验收者。

### 可能形成的产物

现状基线、风险边界、最小自动化能力模型、受控样本实验、人工复核工作流、效率与质量指标、交付文档和回滚方案。

### 第二轮继续提示词

```text
公司只允许使用脱敏样本，最终输出必须由采购经理确认。当前每周整理约 6 小时，返工率约 12%。请设计最低风险的试点项目、能力要求、验收指标和本周计划。
```

### 学习后的复盘与重规划提示词

```text
试点把处理时间降低了，但关键字段错误超过阈值，人工复核负担反而上升。这里是测试记录：[内容]。请诊断失败类型，回滚到最早受影响的能力或项目决策，并给出下一版实验。
```
